# 小红书选题池

## 可继续写的方向

- 为什么 MacBar 选择“本地优先”，而不是做云同步
- 图片 OCR 在真实工作流里的 5 个高频场景
- 菜单栏工具最难做的不是功能，而是“不打扰”
- 我为什么把导入/导出配置删掉了
- 为了一个 `Esc` 和复制后关闭窗口，我反复改了几轮
- 为什么 Finder 文件复制也应该被剪贴板工具认真对待
- 做工具产品时，哪些功能其实应该忍住不做
- MacBar 和 Maccy / Paste / Raycast / Alfred 的定位差异

## 选题规则

- 一篇只讲一个核心主题。
- 能从用户体感切入，就不要先从技术切入。
- 如果是“更新进度”类内容，也要收敛成一个明确主题，比如：
  - 更顺手
  - 更快
  - 更稳定
  - 更轻量
